# FlipSolver

一个用于求解点灯 / 翻转类棋盘小游戏的小工具。

这个项目提供了一个简单的 `PyQt5` 图形界面，可以输入棋盘大小、手动设置初始状态，并计算一组可行点击方案。除经典模式外，还支持我为了游戏关卡额外做的“异形模式”。

## 项目功能

- 支持 `n x n` 方阵棋盘
- 支持经典模式求解
- 支持带不可点击格子的异形模式求解
- 支持随机生成经典关卡和随机生成异形关卡
- 使用 `GF(2)` 上的高斯-约旦消元求解翻转方程组
- 提供图形界面输入和结果显示
- 提供重新设计的双栏界面、更清晰的状态提示和可复制的求解结果

## 游戏规则

### 经典模式

点击某个格子时：

- 该格子本身会翻转
- 该格子所在整行会翻转
- 该格子所在整列会翻转

黑白状态按二值处理：

- `0` 表示白色
- `1` 表示黑色

求解目标是找到一组点击位置，使当前棋盘满足目标状态。

### 异形模式

异形模式用于处理带“不可操作区域”的关卡：

- 在开始前，黑色格子会被视为有效区域
- 非黑色格子会被锁定为灰色，不参与点击和求解
- 求解时会自动删除对应无效格子在方程组中的行列

## 算法思路

项目把每一次点击对棋盘造成的影响写成一个二元线性方程组：

- 每个格子是否点击，记为一个未知数
- 每次点击对整行、整列和自身的翻转效果，写入系数矩阵
- 最终用 `GF(2)` 上的高斯-约旦消元求解

项目结构：

- `main.py`：应用入口
- `flipsolver/gui.py`：图形界面与交互逻辑
- `flipsolver/levels.py`：随机经典/异形关卡生成
- `flipsolver/matrix.py`：构造经典模式和异形模式的系数矩阵
- `flipsolver/solver.py`：在 `GF(2)` 上解线性方程组

## 运行方式

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动程序

```bash
python main.py
```

## 使用说明

1. 输入网格大小 `n`
2. 点击“生成空白棋盘”
3. 通过点击格子设置初始状态（异形模式下会在白色 / 黑色 / 灰色禁用之间循环）
4. 选择模式
5. 也可以直接点击“随机生成当前模式关卡”开始一局
6. 点击“开始游玩”进行手动尝试，或直接点击“计算解”查看答案
7. 在界面右侧“求解结果”中查看建议点击坐标，并可使用“复制结果”按钮

## 随机关卡

- 经典模式会随机生成一个保证可解的黑白棋盘
- 异形模式会随机生成可操作区域和对应的可解局面
- 你可以随机出题自己玩，也可以随时点击“计算解”查看答案

## 说明

- 这个项目主要是为一个具体游戏关卡写的求解器，所以规则不是标准的 Lights Out，而是“同行、同列和自身一起翻转”的变体。
- 当方程组存在多解时，程序会返回其中一组可行解。
